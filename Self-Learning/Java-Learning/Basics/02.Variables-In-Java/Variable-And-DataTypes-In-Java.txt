    Data Types - 8 primitive data types
    DataType           Size             Primitive/Reference                 Value

    byte                1                   primitive                -128 ot 127
    short               2                   primitive                -32,768 to 32,767

    long                8                   primitive                -9 quintillion to 9 quintillion
    float               4                   primitive                fractional number up to 6-7 digits

                                                                        Eg: 'f', 'a', 'm'

                                                                        Eg: "Hello, World!"

Differences between Primitive & Reference 
    Primitive Data Types (Fixed/Reserved Data Types)
        - 8 types (boolean, bytes, short, int, long, float, double, char)
        - stores only data.
        - only holds 1 value
        - use of less memory 
        - faster
    Reference Data Types (User-Defined Data Types)
        - unlimited (string is also an reference data types.)
        - stores an address.
        - capability to holds more than 1 values.
        - use of more memory
        - slower
